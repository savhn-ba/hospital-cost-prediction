# Hospital Cost Prediction

This project predicts hospital treatment cost using patient clinical and admission features.

## Features Used
- Age
- Gender
- Key complaints
- Body weight & height
- Pulse & respiration rate
- Past medical history
- Mode of arrival
- Admission type

## Model
Linear Regression with feature scaling.
